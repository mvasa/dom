Logic Calculator main window

Logic Calculator main window consist of the following elements:



  1. Main menu: Calculator, Examples & Help options
  2. Text area for formula diplay
  3. Input controls: Go, Backspace & Clear buttons
  4. Calculator modes: Logic formula evaluator, Deduction evaluator & Normal Form converter
  5. Available operators (depending of the selected mode)
  6. Up to 34 variables or atoms
  7. Output & Debug tabs
  8. Output text area for displaying results
  9. Status bar
